# US Medical Insurance Data Analysis

This project explores the relationships between **age**, **BMI**, **smoking habits**, **gender**, and **insurance charges** in the United States.  
The goal was to understand how lifestyle and personal factors affect medical costs using clean, Python-based data analysis.

---

## Key Insights
- Smokers have **significantly higher** average charges than non-smokers.  
- **Obese smokers** show the highest mean charge overall.  
- Insurance costs increase steadily with **age** and **BMI**.  

---

## Tools and Methods
- **Python**
- **Jupyter Notebook**
- **CSV file processing**
- **Data cleaning, grouping, and average-based comparisons**

---

## Summary
A compact and logical data analysis project focused on exploring patterns in insurance data through pure Python code.
